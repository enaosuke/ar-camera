# AR Camera

カメラ映像の上にスタンプを重ねて撮影し、画像として保存できる Web アプリです。

## URL
https://enaosuke.github.io/ar-camera/

## 機能

- カメラ起動後、スタンプ（フレーム）を画面上に表示
- PC: マウスでスタンプの移動・拡大縮小・回転（ホバーで枠線とハンドル表示）
- スマホ: タッチ＋ピンチで拡大縮小・回転
- シャッターボタンで合成画像をキャプチャ
- 保存モーダルから PNG / JPG（高・中・低画質）でダウンロード

## 技術

- Vue 3 + Vite 5 + Vue Router
- PixiJS 7（カメラ映像・スタンプの描画）
- SCSS（スタイル）
